# Changelog

All notable changes to this project are documented here.

## [0.1.0] - 2026-07-20

### Added

- Compact Windows taskbar display for Codex weekly and five-hour usage limits.
- Reset countdowns, account/plan details, settings, and usage-versus-remaining bar mode.
- Neutral handling when the five-hour limit is not returned.
- Windows packaging with a portable PyInstaller executable.
- Windows CI and tagged release automation.
